I/O Manager Routines

WHQL restricts NDIS drivers—that are not also NDIS-WDM—from calling the following I/O manager routines:

Restricted routine Error message
IoAllocateDriverObjectExtension Do not use this call.
IoAllocateErrorLogEntry Use NdisWriteErrorLogEntry, NdisWriteEventLogEntry instead.
IoAllocateWorkItem Use NdisScheduleWorkItem instead.
IoAttachDevice Do not use this call.
IoAttachDeviceToDeviceStack Do not use this call.
IoBuildAsynchronousFsdRequest Do not use this call.
IoBuildSynchronousFsdRequest Do not use this call.
IoConnectInterrupt Use NdisMRegisterInterrupt instead.
IoCreateDevice Use NdisMRegisterDevice instead.
IoCreateSymbolicLink Do not use this call.
IoCreateUnprotectedSymbolicLink Do not use this call.
IoDeleteDevice Do not use this call.
IoDeleteSymbolicLink Do not use this call.
IoDetachDevice Do not use this call.
IoDisconnectInterrupt This function was designed for a full MAC driver model, which is no longer supported.
IoFreeWorkItem Do not use this call.
IoGetAttachedDeviceReference Do not use this call.
IoGetDeviceInterfaceAlias Do not use this call.
IoGetDeviceInterfaces Do not use this call.
IoGetDeviceObjectPointer Do not use this call.
IoGetDmaAdapter Do not use this call.
IoGetDriverObjectExtension Do not use this call.
IoGetRelatedDeviceObject Do not use this call.
IoInitializeTimer Do not use this call.
IoInvalidateDeviceRelations Do not use this call.
IoInvalidateDeviceState Do not use this call.
IoOpenDeviceInterfaceRegistryKey Do not use this call.
IoOpenDeviceRegistryKey Do not use this call.
IoQueueWorkItem Do not use this call.
IoRegisterDeviceInterface Do not use this call.
IoSetDeviceInterfaceState Do not use this call.
IoStartTimer Do not use this call.
IoStopTimer Do not use this call.
IoWMIWriteEvent This function was designed for a full MAC driver model, which is no longer supported.
IoWriteErrorLogEntry Use NdisWriteErrorLogEntry, NdisWriteEventLogEntry instead.
